Water Level Monitor
HC-SR04 measures tank depth. Calculates % fill. LCD displays level. Relay auto-controls pump: ON <30%, OFF >90%.
Components
Arduino UnoHC-SR04RelayLCD 16×2LED
Open on Wokwi ↗
1// Water Level Monitor — wokwi.com/projects/457050155667043329 2#include <LiquidCrystal.h> 3LiquidCrystal lcd(7,6,5,4,3,2); 4int trig=9,echo=8,relay=10,led=11; 5long duration; int distance; int tankHeight=20; 6 7void setup() { 8 pinMode(trig,OUTPUT); pinMode(echo,INPUT); 9 pinMode(relay,OUTPUT); pinMode(led,OUTPUT); 10 lcd.begin(16,2); lcd.print("Water Controller"); 11 delay(2000); lcd.clear(); 12} 13void loop() { 14 digitalWrite(trig,LOW); delayMicroseconds(2); 15 digitalWrite(trig,HIGH); delayMicroseconds(10); 16 digitalWrite(trig,LOW); 17 duration=pulseIn(echo,HIGH); 18 distance=duration*0.034/2; 19 int pct=(tankHeight-distance)*100/tankHeight; 20 lcd.setCursor(0,0); lcd.print("Level: "); lcd.print(pct); lcd.print("% "); 21 if(pct<30){ digitalWrite(relay,HIGH); lcd.setCursor(0,1); lcd.print("Motor: ON "); } 22 else if(pct>90){ digitalWrite(relay,LOW); lcd.setCursor(0,1); lcd.print("Motor: OFF "); } 23 delay(1000); 24}

LDR Auto-Brightness
LDR on A0 reads ambient light (0–1023). Inverted mapping to PWM (0–255) on D9. Bright room → dim LED. Dark room → full brightness.
Components
Arduino UnoLDR / PhotoresistorLED10kΩ220Ω
Open on Wokwi ↗
1// LDR Auto-Brightness — wokwi.com/projects/458379892087600129 2const int ldrPin=A0; // Photoresistor 3const int ledPin=9; // PWM output pin 4 5void setup() { 6 pinMode(ledPin,OUTPUT); 7 Serial.begin(9600); 8} 9void loop() { 10 int lightValue=analogRead(ldrPin); // 0–1023 11 // Invert: bright room → LED dim, dark room → LED bright 12 int brightness=map(lightValue,0,1023,255,0); 13 brightness=constrain(brightness,0,255); 14 analogWrite(ledPin,brightness); 15 Serial.print("Light Sensor Value: "); Serial.print(lightValue); 16 Serial.print(" | LED Brightness: "); Serial.println(brightness); 17 delay(200); 18}

02 — Subject Details

The science of Robotics

Robotics combines electronics, programming, and system design to build machines that sense and respond to their environment.

All five projects demonstrate distinct robotics sub-skills — from binary sensing and PWM proportional control to multi-input button debouncing and automated feedback loops.

ArduinoC++ SensorsPWM AutomationWokwi
MOD 01
Digital Sensing
PIR sensors (P1, P2) produce HIGH/LOW signals — the foundation of any presence-detection or security system.
MOD 02
Analog & PWM Control
LDR→analogRead→map→analogWrite pipeline (P4). Input_PULLUP button→PWM fan speed control (P5). Both demonstrate proportional output.
MOD 03
Ultrasonic Distance
HC-SR04 timed pulse technique: distance = duration × 0.034 / 2. Applied in P3 for real-time tank level calculation.
MOD 04
Relay Automation
Threshold-based bang-bang control in P3 — motor ON <30%, OFF >90%. Classic embedded automation pattern.
